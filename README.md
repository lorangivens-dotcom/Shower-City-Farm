# Meteor Shower City & Farm

Meteor Shower City & Farm is an interactive creative coding project built in Processing. The program displays an animated meteor shower over two different environments: a city scene and a farm scene. Users can switch between scenes by pressing the space bar.

## Features
- Animated meteor shower
- City scene with buildings and road
- Farm scene with barn, fence, and trees
- Scene switching with keyboard input
- Night sky with stars and moon

## Built With
- Processing
- Java-based programming concepts

## What I Learned
This project helped me practice:
- Arrays
- Animation logic
- Conditional statements
- Scene design
- User interaction
- Debugging and organizing code

## How to Run
1. Open Processing
2. Create a new sketch
3. Paste the code into the sketch
4. Click Run
